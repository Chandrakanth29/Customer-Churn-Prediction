# 📉 Customer Churn Prediction

An end-to-end machine learning project that predicts customer churn using the IBM Telco Customer Churn dataset. The notebook covers data preprocessing, exploratory analysis, model training, evaluation, and churn prediction using Logistic Regression and Random Forest.

---

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to leave so that retention strategies can be planned.

This project demonstrates a complete machine learning workflow using Python and scikit-learn.

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / Google Colab

---

## 📊 Models Used

- Logistic Regression
- Random Forest Classifier

---

## 📈 Workflow

1. Load customer churn dataset
2. Clean and preprocess the data
3. Encode categorical features
4. Scale numerical features
5. Split the dataset into training and testing sets
6. Train Logistic Regression and Random Forest models
7. Evaluate model performance
8. Generate churn predictions
9. Visualize the confusion matrix

---

## 📂 Dataset

This project uses the **IBM Telco Customer Churn** dataset.

Download:

https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Save the dataset as:

```text
churn_data.csv
```

Place it in the same folder as the notebook before running it.

---

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Open and run:

```text
Customer_Churn_Prediction.ipynb
```

Run all cells from top to bottom.
